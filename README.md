# Middleware

這是 AC 2-3 學期 A13 用 Node.js、Express 將所有需求回應紀錄在 sever log 中。

## 環境建置與需求 (prerequisites)

- Visual Studio Code - 開發環境
- express - 4.17.1
- npm - a JavaScript package manager
- nvm version 0.38.0


## 安裝與執行步驟

1. 打開終端機，clone 此專案至本機電腦
   `https://github.com/shijia22/AC_2-3_middleware.git`
2. 開啟終端機，進入存放此專案資料夾
   `cd AC_2-3_middleware`
3. 安裝 npm 套件
   `npm install`
4. 啟動專案
   `npm run dev`
5. 出現以下訊息後，即可在 http://localhost:3000 開始使用
   `Express is listening on localhost:3000`
